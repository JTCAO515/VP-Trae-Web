# 数据迁移目录说明

本目录用于存放 VisePanda 平台数据库迁移脚本、初始化种子与回滚辅助脚本。

## 约定

- 使用递增前缀管理迁移顺序，例如 `001_auth_init.sql`
- 一个迁移文件只处理一个领域的结构变更，避免跨域耦合
- 迁移脚本默认面向全新数据库，不兼容旧库结构

## 推荐目录内容

- `00x_<domain>_init.sql`：领域初始化建表脚本
- `seed/`：本地或测试环境所需的基础种子数据
- `rollback/`：明确可逆的回滚脚本

## 本地执行入口

当前阶段建议通过 API 工程或后续统一脚本触发迁移。最小执行方式如下：

```bash
psql "$DATABASE_URL" -f infra/migrations/001_auth_init.sql
```
